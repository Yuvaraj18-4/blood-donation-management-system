Blood Donation Web App

The **Blood Donation Web App** is a modern web-based platform built using **React.js** to connect **blood donors** and **recipients** efficiently.  
It simplifies the process of finding donors nearby, registering new donors, and managing blood donation data — ultimately helping to save lives through a user-friendly and fast interface.

Features

- Donor Registration & Management  
- Search Donors by Blood Group & Location  
- Real-time Map Integration for Nearby Donors  
- Emergency Request System  
- Interactive User Interface built with React Components  
- Fully Responsive Design for all devices
