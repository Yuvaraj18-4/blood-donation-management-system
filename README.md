Blood Donation Web App

The **Blood Donation Web App** is a modern, web-based platform built with **React.js** to efficiently connect blood donors and recipients.  
It simplifies finding donors nearby, registering new donors, and managing blood donation data — ultimately helping to save lives through a fast and user-friendly interface.

Project Overview

This project aims to create a digital platform that bridges the gap between blood donors and recipients.  
Users can register as donors, search for donors by **blood group or location**, and request help during emergencies.  
The app ensures **real-time access**, **ease of use**, and a **responsive design** for all devices, making it suitable for hospitals, organizations, and individual donors

Features

- Donor Registration & Management  
- Search Donors by Blood Group & Location  
- Real-time Map Integration for Nearby Donors  
- Emergency Request System  
- Interactive User Interface built with React Components  
- Fully Responsive Design for Desktop and Mobile Devices
